# Example Python client for Open Ephys Network Events

This folder contains three example Python scripts that illustrate the functionality of the
Network Events plugins.
- [`record_control_example_client.py`](record_control_example_client.py): Remote controls data acquisition in Open Ephys.
- [`network_events_console.py`](network_events_console.py): Manually send commands to Open Ephys via ZeroMQ.
- [`test_ttl_events.py`](test_ttl_events.py): Starts a recording and sends many `TTL Line` and `TTL Word` commands.
